Adyen Python API

This is the documentation on how to install and use the Adyen Python API. The API is a Python package which can be installed by downloading the latest release from Github, adding an import statement and creating an instance of the 'Adyen' class.

These docs are written to correspond with the Adyen general documentation. We recommend to read the general docs first and keep a tab open while you use this guide.

This guide assumes you have a test merchant account with a webservice user created, and a 'hosted payment pages' skin configured. If you do not yet have a merchant account sign up here.

Installation

Requirements:

  • Python 2.7
  • The library is not yet compatible with Python 3

Optional:

  • requests
  • pycurl
  • If neither is installed, the library will use urllib2 by default

Import

  1. Place the Adyen library directory in your server directory.
  2. Import Adyen package in your Python code.
  3. Create an instance of the Adyen.Adyen class.

Creating an Adyen class instance

Initialise a class instance, and set the attributes shown below with your credentials and 'hosted payment pages' information. 

              
  Import Adyen

  ady = Adyen.Adyen()

  ady.payment.client.username = "web service user name"
  ady.payment.client.skin_code = "hosted payment pages skin code"
  ady.payment.client.hmac = "hmac key associated with this skin generated in the customer area."
  ady.payment.client.platform = "test or live"
  ady.payment.client.merchant_account = "merchant account name"
  ady.payment.client.password = "web service user password"
  ady.payment.client.app_name = "Your unique app name, to identify the app with Adyen."

The example connects you to the Adyen test platform. If you want to use our live platform use: 

            
              adyenbase.payment.client.platform = "live"

Now the Adyen Python library is ready to process and manage payments with the Adyen platform.
